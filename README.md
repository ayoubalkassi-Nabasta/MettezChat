# 💬 MettezChat - تطبيق دردشة Android

تطبيق دردشة فوري مكتوب بـ **Kotlin** و **Firebase** ☁️

## ✨ الميزات

- 🔐 تسجيل دخول وإنشاء حساب آمن
- 💬 إرسال واستقبال رسائل فوراً
- 👥 عرض قائمة المحادثات
- 🔔 تنبيهات الرسائل الجديدة
- 📱 واجهة عربية كاملة

## 🛠️ التقنيات المستخدمة

- **Kotlin** - لغة البرمجة
- **Firebase Realtime Database** - قاعدة البيانات
- **Firebase Authentication** - المصادقة
- **Material Design** - التصميم
- **RecyclerView** - عرض القوائم

## 📋 المتطلبات

- Android Studio 4.0+
- SDK 21+
- Firebase Account
